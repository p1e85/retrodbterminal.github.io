## Current Version
v1.0 (Free Version) - Released September 28, 2025

This is the complete, stable, local-only version of the application. It includes all features and bug fixes up to our most recent update.

UI & Theming: Fully responsive mobile design with four selectable retro themes (Green, Amber, Vaporwave, Monochrome) and a theme cycler.

Note Management: Create, edit, and delete notes. A checkbox system allows for selecting single or multiple entries.

Data Handling: All notes are saved to the browser's local storage.

Exporting: Users can download single notes or a combined batch file for multiple selections.

Meta Features: Includes an "Info" modal explaining the privacy-focused local storage, a "Support" modal with a PayPal link, and full on-page SEO tags.
